# Palvelin-projekti
Projekti, jossa teemme Saltilla Windows 11 mikrofonin ääniasetuksista idempotettisia.


## Windows 11 VM luonti 
Asennettiin Windows 11 25H2 VirtualBoxiin ilman päivityksiä UUPDUMPIN sivulta ja tarkistettiin, että siellä olisi päivityksiä tekemättä jotta pystymme simuloimaan samnlaista tilannetta.
